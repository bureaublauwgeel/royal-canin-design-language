Carousels

Usage and implementation for Carousels within Web Design Language.

Introduction

A carousel should be used in an area of significance and is built up of an image, title, button, pager and navigation icons. Several carousel options are available below to ensure each layout can be accommodated.

To display a carousel, use the following general strucutre.

Each div inside of rc-carousel constitues a slide. For RC Carousels, each slide is referred to as a hero and can use 1 of 4 layouts.

The data attribute must be included with ecah hero.


					@@include('./03-features/carousels/_carousel-structure.html')

A carousel hero can use 1 of 4 layouts. The layout used is determined by which modifier class is added alongside the block hero class. There are 3 modifier classes available.

All Carousel Heros use the same element classes inside the hero block, but in different amounts and orders.

  • The hero__bg class should be applied to a background image or a video.
  • The hero__fg class wraps any foreground content.
  • The hero__section block and moidifer classes should be applied to content inside the foreground.
  • A standard anchor with the button button--icon-arrow-right classes is provided for call to actions.
