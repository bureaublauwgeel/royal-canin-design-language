Headings - Modifiers

Usage and implementation for Typography within Web Design Language.

Introduction

Wherever possible, when a Heading tag is used, an accompanying heading class must be applied to set it's appearance to one of the avaiable options demonstrated below. These classes should be used in any Component, Feature or Template to set the appearance of a Heading.

This is due to the HTML5 document outline algorithm allowing multiple content roots on any HTML5 marked up webpage. This means, in short, that multiple <h1> tags can be used if separated by the appropriate HTML5 semantic sectioning elements.

The use of an rc-head--[value] class will set the appearance of any Heading tag, or indeed any element with that class applied to it.

As demonstrated brlow, a rc-head--[value] modifier class will override the default font size of a standard Heading tag.
