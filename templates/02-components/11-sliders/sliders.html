Sliders

Usage and implementation for Sliders within Web Design Language.

Introduction

A slider should be used to select a value where multiple answers could be an option, such as an age picker. Icons to use in the pip can be selected from the icon library. Sliders will work from mobile devices up and will feedback to a user the option they have selected instantly.

The slider relies on nouislider. Please refer to the documentation for more information.
