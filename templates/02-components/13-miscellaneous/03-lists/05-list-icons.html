Icon List

Usage and implementation for Lists within Web Design Language.

Introduction

Lists items can have an extra modifier class added to them in order to display an SVG icon at the start. The root class list-icon must be applied, then the actual icon class must be added. Pick from:

  • list-icon--check
  • list-icon--arrow-down
  • list-icon--arrow-right
