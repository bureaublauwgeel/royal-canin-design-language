Social

Usage and implementation for Sharing section within Web Design Language.

Introduction

Social icons should be displayed in social brand colours where possible. If the nature of the design does not allow for social brand colours Royal Canin brand colour options are available. NB These are suggested layouts, all social icons can be found in gthe icon set. The main structure and styling is consistent between these variations, only the section label will change to indicate the purpose to the user. When using the Social section as a way to link to Royal Canin social networks, please ensure you have the correct links for your Country's social media accounts.

Examples

The examples below display front end styling and HTML markup to implement the Sharing section into your code.
