Logos

Introduction

Brand logos are available for use in any project, the guidelines below will explain correct logo use throughout any project.

Use of the main logo is required when the strapline is legible. It is recommended that the primary logo be used when the height is no lower than 60pt.

    @@include('./02-components/01-logos/_logo-primary.html')

Use of the primary logo with no strapline is suggested for when the logo is required to have a height of smaller than 60pt. It is recommended that the height of the logo should not be lower than 20pt.

    @@include('./02-components/01-logos/_logo-secondary.html')

To only be used as a secondary option, where the main logo has previously been presented and the user has already been acquainted with the brand.

    @@include('./02-components/01-logos/_logo-crown.html')
