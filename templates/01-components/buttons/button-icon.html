Buttons - Icon

Usage and implementation for Buttons within Web Design Language.

Introduction

Icons can be added to Buttons with the addition of a modifier class.

Seocondary and Tertiary modifier classes will automatically change the colour of the Button's Icon.

The available icon classes are demonstrated below.
