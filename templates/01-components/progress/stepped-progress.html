Stepped Progress

Usage and implementation for Progress Bars within Web Design Language.

Introduction

The Progress element is used to display a known percentage between two values.

If the browser does not support the <progress> element (IE9), then the fallback div will be shown instead.

This component relies on accompanying Javascript in order to display the current value between the min and max values, as well as the ability to update that value.
