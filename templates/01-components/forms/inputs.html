Inputs

Available inputs for Forms.

A note on accessibility.

Please ensure that the for="" attribute is used on labels and that it matches the input ID, for accessibility purposes, for example: 

@@include('./01-components/forms/_input-standard.html', {
				"type": "text" 
			})

Since the for attribute relies on an ID to match up with, ensure that all Input IDs are unique on your page.

Type - Checkbox

When using Checkboxes, ensure that the accompanying Label either wraps the input, or is place directly after it.

What follows is an example of both implementations.

In a form or a real world situation, Checkboxes and their labels should be grouped inside of, and positioned by, a containing element.
