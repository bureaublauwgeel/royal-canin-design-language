Social

Usage and implementation for Sharing section within Web Design Language.

Introduction

Social network links can be used as either links to Royal Canin's social network accounts, or as a way to share a particular post or page on a user's account.

The main structure and styling is consistent between these variations, only the section label will change to indicate the purpose to the user.

When using the Social section as a way to link to Royal Canin social networks, please ensure you have the correct links for your Country's social media accounts.

Examples

The examples below display front end styling and HTML markup to implement the Sharing section into your code.
