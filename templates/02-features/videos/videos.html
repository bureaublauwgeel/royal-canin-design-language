Videos

Usage and implementation for Videos within Web Design Language.

Introduction

Videos should be displayed using embed code from a source like YouTube or Vimeo.

The embed code should be placed inside a predetermined class to ensure the element is responsive.

Examples

The examples below display front end styling and HTML markup to implement Videos into your code.
