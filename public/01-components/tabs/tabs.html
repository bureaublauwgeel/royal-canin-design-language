Tabs

The Tabs Component relies on the main Royal Canin WDL Javascript File.

Whilst this component is flexible enough to accommodate any number of tabs and any amount of content in the content boxes, it is recommended than when using it, you follow these guidelines:

  • No more than 5 tabs.
  • Trigger button values should be short.
  • Trigger button SVGs should be relevant and used for all buttons, or none.
  • Images used inside the content blocks must use the align classes, outlined here.
  • All Trigger buttons must contain text as well as an SVG, or an SVG by itself.
  • If the Trigger buttons contain no text (just an SVG), the rc-tabs--icons class must be applied in addition to the wrapping rc-tabs class.
  • The clearfix class should not be removed.
