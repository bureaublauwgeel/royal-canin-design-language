Welcome

With over 50 global websites built across various content management systems managed by a number of different regions and local agencies, a consistent set of digital styles has been developed. To ensure all designers and developers working with the Royal Canin brand begin any project with a consistent set of tools which meet the exacting standards required on any Royal Canin project the Web Design Language is an open source project aimed at contunually evolving through input from user and region.

This approach aims to bridge the gap between brand style guides and the practicalities of allying that to the web. The intention is to remove ambiguity for all the core components that are common to any website in any region.

Direction

The Web Design Language is intended to give direction, but also be a collaboration of the family of designers and developers working globally on behalf of the Royal Canin brand, all in the pursuit of delivering a magnificent user experience.
