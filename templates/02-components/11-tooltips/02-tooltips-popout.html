Tooltips - Popout

Usage and implementation for Tooltips within Web Design Language.

Introduction

Tooltips should be used primarily to display additional content that is beneficial yet not crucial to the user.

Tabbed Tooltips are available but the content that is displayed in them should always primarily be presented in a way that does not requir user interactivity to reveal.

To put it simply, the main content should not rely on the Tooltip in order to be coherent.
