Sliders

Usage and implementation for Sliders within Web Design Language.

Introduction

Relies on nouislider. Please refer to the documentation for more information.

Sliders are an interactive means to enable a user to select a value when a timescale for selection is useful.

Sliders will work from mobile devices up and will feedback to a user the option they have selected instantly.

The Sliders described here are not to be mistaken with the Carousels Feature.
