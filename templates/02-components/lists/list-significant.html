Signficant List

Usage and implementation for Lists within Web Design Language.

Introduction

Lists can be implemented with an icon set pre defined using SVG icons.

Lists can contain external links built inline.

The CSS class list--significance uses a Sass @extend rule to inherit the properties of the list--blank and list--align classes as standard
