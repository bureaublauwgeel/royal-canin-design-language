Image Galleries

Usage and implementation for Image Galleries within Web Design Language.

Introduction

An image gallery can be used to display a selection of related images that the user can scroll through. Image thumbnails are optional.

It is recommended that, if using image thumbnails, hard croppped consistently sized square images are used.

Please note, this Feature requires Javascript configuration, as well as strict class and ID naming conventions.
