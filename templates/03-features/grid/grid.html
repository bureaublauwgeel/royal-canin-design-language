Grid

Usage and implementation for Grid within Web Design Language.

Introduction

The Web Design Langauge grid system is made up of 5 Column options. Columns counts can be mixed and variations documented below will enable a wide range of column options.

Grid column layouts can be mixed to suit your content as long as the sum of the fractional value is 1.

A new row should be used each time the fractional value equals 1.
