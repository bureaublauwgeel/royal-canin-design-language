Icons

Usage and implementation for Icons within Web Design Language.

Introduction

Icons are used throughout the Web Design Language to further enhance the UI and assist a user on their journey. Icons can be used within buttons by adding the relevant class and all icons are sized equal to ensure they remain consistent within the button. Icons can also be implemented as an isolated element as part of any content by copying and pasting the relevant markup into your document from the portal.

The Icons displayed below have been prepared and optimised to use across Royal Canin touchpoints as a set. If an icon that you require does not exist please create an issue and describe the icon you require.
