Typeface

Usage and implementation for Typography within Web Design Language.

Introduction

Din Family is used throughout Web Design Language projects with a variety of weights and sizes used to convey a message to the user. Brand red, text colour and white are the only available colours to use in generic copy.

Examples

  • Din Pro Regular

    Din Pro Regular is implemented through standard <p>, <li>, <span> and <label> tags.

    To implement Din Pro Regular, add the class font--regular

    ABCDEFGHIJKLMNOPQRSTUVWXYZ
    abcdefghijklmnpqrstuvwxyz
    123456789

  • Din Pro Medium

    Din Pro Medium is implemented through standard <h>tags.

    To implement Din Pro Medium add the class font--medium

    ABCDEFGHIJKLMNOPQRSTUVWXYZ
    abcdefghijklmnpqrstuvwxyz
    123456789

  • Din Pro Bold

    Din Pro Bold should only be implemented in extreme cases as it can appear overpowering to the user.

    To implement Din Pro Bold add the class font--bold

    ABCDEFGHIJKLMNOPQRSTUVWXYZ
    abcdefghijklmnpqrstuvwxyz
    123456789
