Tables

Usage and implementation for Tablies within Web Design Language.

Introduction

Tables should be used when presenting information that is reuired to be presented in a structured template.

Table content can be left or right aligned and can contain standard text or form elements.

Colspan and Rowspan classes can be added to span content.

The default styling will make the tables scroll horizontally when the browser window is narrower than the data being displayed.

General Notes

The following attributes are now deprecated and should not be used:

  • align
  • bgcolor
  • border
  • cellpadding
  • cellspacing
  • frame
  • rules
  • summary
  • width
