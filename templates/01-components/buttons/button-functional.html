Button - Functional

Usage and implementation for Buttons within Web Design Language.

Introduction

Buttons can be used for a variety of actions.

Sizing modifier classes can be used to change the size of the Button.

Button states can give different visual representations to the user.

Icons can be used within Buttons to hightlight a specific area or action.

Examples

A functional call to action should be used when an interaction is required by the user which will result in them being presented with additional information based on their requirements.
