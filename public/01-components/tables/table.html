Table

Tables should be used when it is appropriate for the data.

The following attributes are now deprecated and should not be used:

  • align
  • bgcolor
  • border
  • cellpadding
  • cellspacing
  • frame
  • rules
  • summary
  • width

The default styling will make the tables scroll horizontally when the browser window is narrower than the data being displayed.


Structure

The thead, tbody and tfoot tags must be used where appropriate.
