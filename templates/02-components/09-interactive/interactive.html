Interactive

Usage and implementation for Buttons within Web Design Language.

Introduction

These components are to be used when interaction with a Feature is required, such as Carousel pagination, or deciding Yes or No.

Examples

Success and error icons are used when a user has interacted with an icon and is to be returned a visual result.

Use this markup where possible for an individual link, or use this Component's CSS in order to style an individual pagination links whose structure can't be customised.
