Carousel Heros

Usage and implementation for Carousels within Web Design Language.

Introduction

A carousel should be used in an area of significance and is built up of an image, title, button, pager and navigation icons. Several carousel options are available below to ensure each layout can be accommodated.

A carousel hero can use 1 of 4 layouts. The layout used is determined by which modifier class is added alongside the block hero class. In the example below, each of the layouts is displayed sequentially, beginning with the block hero class without a modifier.

Carousel Version 3
Single lower image, introduction text and call to action.

Mobile Layout - Pet or packshot imagery should be full width. Introductory text should display above the image. Call to action should display below the introductory text.
Tablet Layout - Pet or packshot imagery should be full width below the introductory text. Introductory text should display above the image. Call to action should display below the introductory text.
Desktop Layout - Pet or packshot imagery should be full width below the introductory text. Introductory text should display above the image. Call to action should display below the introductory text.
