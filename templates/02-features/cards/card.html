Card - Default

Usage and implementation for Cards within Web Design Language.

Introduction

Cards are used to present the user with a link to a specific page or post.

Each Card's image should be presented as background image, to ensure consistent sizing across all cards.

Because of this, the <figure> element that presents the image must have role, title and aria-label attributes to aid in accessibility.
